# sln2gyp [![Build Status](https://travis-ci.org/kbinani/sln2gyp.svg?branch=master)](https://travis-ci.org/kbinani/sln2gyp)

sln2gyp is a CLI tool to create .gyp files from a VisualStudio solution file (*.sln).

# Usage

```shell
sln2gyp sample.sln
```

# Requirement

Python 2.7
